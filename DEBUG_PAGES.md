# Debug pages

I made these pages for analyzing live data from the game. These can be useful for troubleshooting, or for making your own overlays.

- Header Debug (sent by any packet)
    - http://localhost:3000/debug/header-debug
- Session Packet Debug (ID 1)
    - http://localhost:3000/debug/session-debug
- Lap Data Packet Debug (ID 2)
    - http://localhost:3000/debug/lap-data-debug
- Event Packet Debug (ID 3)
    - http://localhost:3000/debug/event-debug
- Participants Packet Debug (ID 4)
    - http://localhost:3000/debug/participants-debug
- Car Telemetry Packet Debug (ID 6)
    - http://localhost:3000/debug/car-telemetry-debug
- Car Status Packet Debug (ID 7)
    - http://localhost:3000/debug/car-status-debug
- Car Damage Packet Debug (ID 10)
    - http://localhost:3000/debug/car-damage-debug
- Session History Packet Debug (ID 11)
    - http://localhost:3000/debug/session-history-debug
- Time Trial Packet Debug (ID 14)
    - http://localhost:3000/debug/time-trial-debug
