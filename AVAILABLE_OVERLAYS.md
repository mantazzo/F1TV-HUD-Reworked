# Available Overlays

Use these overlays in recording software (e.g. OBS). Size is described in (Width x Height) format.

- Speedometer
    - http://localhost:3000/speedometer
    - Size: 500 x 500
- Lap Timer
    - http://localhost:3000/lap-timer
    - Size: 480 x 270
- Pit Timer
    - http://localhost:3000/pit-timer
    - Size: 340 x 220
- Live Speed
    - http://localhost:3000/live-speed
    - Size: 310 x 365
- Fastest Lap (popup)
    - http://localhost:3000/fastest-lap
    - Size: 920 x 125
- Weather
    - http://localhost:3000/weather
    - Size: 280 x 725
